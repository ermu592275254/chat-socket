# 基于webSocket的聊天后台程序

此程序为我的另一个项目的后台：https://github.com/ermu592275254/x-chat

## 开始

请确保已安装安装mongodb： https://www.mongodb.com/download-center

开启mongodb服务: http://www.runoob.com/mongodb/mongodb-window-install.html

开启项目
```
npm run dev
```

```
//热替换
npm run hot
```
```
// debugger
npm run debug
```
